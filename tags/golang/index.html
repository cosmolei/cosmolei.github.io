    golang模块开发

    2020-12-09
    Tech
    go, golang

    [[golang]]

    golang模块开发流程：

    1. 创建项目目录

      $ mkdir myapp
$ cd myapp

    2. 模块初始化， 会生成go.mod文件

      $ go mod init myapp

    3. 创建main入口文件

      // 文件名随意，但package要是main
package main

import "fmt"

func main() {
	fmt.Println("Welcome")
}

    4. 引入第三方库之后，执行go mod vendor下载

      package main

import (
	"fmt"
	"gopkg.in/alecthomas/kingpin.v2"
)

func main() {
	kingpin.Parse()
	fmt.Println("Welcome")
}
