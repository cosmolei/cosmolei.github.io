    日常折腾

    在centos环境中，运行rpmbuild，会生成/home/$USER/rpmbuild目录

    # 先随便指定一个文件生成目录，之后再编写spec文件
$ rpmbuild test.spec

├── BUILD
├── BUILDROOT
├── RPMS
├── SOURCES
