    2021-01-13
    Tech
    Openstack

    Ubuntu 20.04 单节点安装部署microstack #

    1. 安装 #

    通过snap安装，如果没有snap请先安装snap。

    执行以下命令安装：

    $ sudo snap install microstack --devmode --beta

    如果安装成功，则跳到第二部分初始化。

    如果安装完提示未成功，systemctl查看status提示

    modprobe: ERROR: could not insert ‘vhost_vsock’: Device or resource busy

    则查看是否有冲突

    $ lsmod | grep vsock
# vsock模块冲突
vmw_vsock_virtio_transport_common    32768  0
vmw_vsock_vmci_transport    32768  1
vsock                  36864  3 vmw_vsock_virtio_transport_common,vmw_vsock_vmci_transport
vmw_vmci               69632  2 vmw_balloon,vmw_vsock_vmci_transport
    ...
