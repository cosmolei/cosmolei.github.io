    brew install tmux

    Tmux顺手快捷键 #

    常用快捷键列表
    创建窗格（左右分隔） Ctrl + b %
    创建窗格（上下分隔） Ctrl + b "
    光标切换窗格 Ctrl + b
    切换到上一个窗格 Ctrl + b ;
    切换到下一个窗格 Ctrl + b o
    调整窗格大小 Ctrl + b Alt
