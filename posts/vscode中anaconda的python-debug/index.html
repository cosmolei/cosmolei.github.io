    vscode中anaconda的python debug

    2021-04-15
    Tech
    python, vscode, debug

    在调试ioserver的过程中，发现并没有正常条转到断点。

    原因是在conda的环境ioserver中，安装了项目打包后的程序，导致从.vscode目录下的settings.json中配置的环境变量PYTHONPATH的效果抵消了。

    于是卸载了项目打包后的程序，之后，重新正确配置PYTHONPATH

    {
    "python.pythonPath": "/Users/cosmo/miniconda3/envs/the_env/bin/python",
    "terminal.integrated.env.osx": {"PYTHONPATH":"${workspaceFolder}"}
}

    解决问题。
