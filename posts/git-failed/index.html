    git clone时遇到fatal: unable to access server certificate verification failed #

    git clone 时遇到这样的问题：

    fatal: unable to access ‘https://*.git’: server certificate verification failed. CAfile: none CRLfile: none

    解决办法：

    git config --global http.sslverify false
