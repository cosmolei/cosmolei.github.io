    搭建私有npm仓库

    2021-03-22
    Tech
    nodejs, npm, verdaccio

    发布到npm仓库（本地使用verdaccio搭建私有npm仓库） # 

    # 全局安装verdaccio
npm install -g verdaccio

# 启动verdaccio
verdaccio

# 创建verdaccio用户
npm adduser --registry http://localhost:4873

# 发布包
npm publish --registry http://localhost:4873
