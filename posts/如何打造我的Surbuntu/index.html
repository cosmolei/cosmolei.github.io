    零、基础准备 #

    1. 开启ssh服务 # 

    sudo apt install ssh
# ssh安装后默认开启sshd服务并加入开机启动，如果没有，查看systemctl status sshd
# 非enabled状态的话执行systemctl enable sshd
# 如果不要求开机启动的话，就不需要做这一步。

    2. 安装VIM # 

    sudo apt install vim

    3. 载入维护的/etc/hosts文件 #

    不要问为什么，知道的自然知道，不知道本不需要知道。

    4. 安装git # 

    sudo apt install git

    基础配置一下git：

    全局保存密码

    git config --global credential.helper store

    证书问题

    git config --global http.sslVerify false

    设置邮箱、姓名

    git config --global user.email "you@example.com"
git config --global user.name "Your Name"

    不显示中文，却显示八进制数字的问题

    git config --global core.quotepath false

    1）unlock login keyring #

    1. Press Alt+F2, type seahorse and press Enter to start the Gnome Keyring Manager:

    2. Alternately, open a terminal with Ctrl+F2+T, type seahorse & and press Enter.

    3. The “Passwords and Keys” window should come up as shown below. Under the Passwords tab, select login, right-click on it, and then click on Change Password:

    4. The “Change Keyring Password” box will come up. Type your old password, and then leave the new/confirm password fields blank. Then press OK, and the information box shown below will pop-up; read it, and then click on Use Unsafe Storage to not have to enter your password at each login:

    5. Close the keyring manager. After you log out/reboot, you won’t be asked for your password any more.

    3）一些alias #

    习惯了在mac下用open .来打开Finder，ubuntu下默认Files应用是nautilus。编辑环境变量文件把nautilus变成open。

    一、打造Shell #

    1. oh-my-zsh #

    安装oh-my-zsh # 
    sudo apt update
sudo apt install curl
sudo apt install zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
    安装必要插件 #

    在$HOME/.oh-my-zsh/custom/plugins目录下clone

    git clone git://github.com/zsh-users/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting.gi

    修改~/.zshrc文件

    plugins=(git
        zsh-autosuggestions
        zsh-syntax-highlighting)
    更改主题：powerlevel10k #

    下载主题

    git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
# 国内可以使用gitee上的官方镜像
git clone --depth=1 https://gitee.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

    修改～/.zshrc文件

    ZSH_THEME="powerlevel10k/powerlevel10k"

    安装缺少的字体

    为了减少折腾并且减少空间，统一使用MesloLGS NF系列的Nerd Font，也是powerlevel10k库提供的字体文件：

    MesloLGS NF Regular.ttf

    MesloLGS NF Bold.ttf

    MesloLGS NF Italic.ttf

    MesloLGS NF Bold Italic.ttf

    下载到本地双击安装即可

    4. emacs #

    由于ubuntu20.04源带的emacs版本太低，于是选择编译安装emacs最新版

    下载源码包

    wget http://mirror.team-cymru.com/gnu/emacs/emacs-27.2.tar.gz
tar zxvf emacs-27.2.tar.gz
cd emacs-27.2

    准备编译环境

    sudo apt install autoconf make gcc texinfo libgtk-3-dev libxpm-dev libjpeg-dev libgif-dev libtiff5-dev libncurses5-dev libxml2-dev libgnutls28-dev

    运行./autogen.sh

    ./autogen.sh

    ./configure

    make -j 2
sudo make install

    8. fusuma #

    触控板手势配置程序。

    IMPORTANT: You MUST be a member of the INPUT group to read touchpad by Fusuma.

    $ sudo gpasswd -a $USER input

    Then, You apply the change with no logout or reboot.

    $ newgrp input

    1. Install libinput-tools

      You need libinput release 1.0 or later.

    $ sudo apt-get install libinput-tools

    1. Install Ruby

      Fusuma runs in Ruby, so you must install it first.

    $ sudo apt-get install ruby
    1. Install Fusuma
    $ sudo gem install fusuma

    1. Install xdotool (optional)

      For sending shortcuts:

    $ sudo apt-get install xdotool

    1. golang环境 #

    官网下载go的压缩包

    rm -rf /usr/local/go && tar -C /usr/local -xzf go1.14.3.linux-amd64.tar.gz

    export PATH=$PATH:/usr/local/go/bin

    配置代理

    go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct
